# 💪 Fitness Tracker

Aplicación web para controlar el peso diario durante la rutina de ejercicios y hacer seguimiento semanal.

## Características

- 🔐 **Autenticación**: Cuenta separada para cada usuario
- 📝 **Registro fácil**: Ingresa ejercicios y pesos por día
- 📊 **Gráficos**: Evolución del peso y distribución de ejercicios
- 📈 **Resumen**: Entrenamientos completados vs objetivo
- 👥 **Comparación**: Ve el avance del otro usuario
- 💾 **Datos locales**: Todo se guarda en el navegador

## Instalación

```bash
# Clonar o descargar el proyecto
git clone <tu-repositorio>
cd fitness-tracker

# Instalar dependencias
npm install

# Correr en desarrollo
npm start
```
